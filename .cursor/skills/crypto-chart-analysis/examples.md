# Примеры разборов

## Пример 1: SPCXUSDT · Binance · 5m (Coinglass 4-panel)

**Вход:** скрин с ценой ~160.06, крупный спайк long liq ~14:30 (>80K), CVD −19.84K, OI ~1.251M.

### Вердикт
**SHORT (осторожно) / WAIT для ретеста** · уверенность **6/10** · горизонт **15–60 мин**

### Контекст рынка
- HTF: не виден → локальный intraday разбор
- Фаза: `correction_up` внутри дневного range / возможный `consolidation` 159.5–160.3
- OI-нарратив: после liq OI слегка восстановился → смешанный; ранее `long_unwind` на падении

### Что видно на графиках
1. **Цена:** range 159.40–160.30; текущая ~160.06 — середина-верх диапазона; после дампа отскок без нового HH
2. **Ликвидации:** мощный кластер long liq ~14:30 → капитуляция; типичная зона поддержки ~159.50
3. **CVD:** устойчивый нисходящий «ступенчатый» профиль (−19.84K) → агрессивные продажи доминируют весь день
4. **OI:** падение с 1.254M к 1.246M на liq, затем лёгкий рост → часть позиций перезашла; без явного squeeze

### Конвергенция
⚠️ Частичная: CVD↓ + дневная слабость = bearish bias, но цена уже отскочила от liq zone — **плохой R:R для market short в середине range**

### Уровни
| Уровень | Цена | Роль |
|---------|------|------|
| R1 | 160.25–160.30 | Хай сессии, сопротивление |
| R2 | 160.10 | Зона stop-hunt перед liq |
| S1 | 159.50 | Дно + post-liq support |
| S2 | 159.40 | Абсолютный low окна |

### Сценарии
**Базовый SHORT:** ретест 160.15–160.25 (LH) + отказ 5m + CVD не растёт → TP1 159.70, TP2 159.50, стоп 160.35  
**Альтернатива LONG:** только при пробое 160.30 + CVD разворот вверх + OI↑ → цель 160.50+  
**Инвалидация short:** закрытие 5m выше 160.35

### План сделки
- Вход: limit 160.18–160.22 или после bearish 5m pin на R1
- Стоп: 160.36 (~0.1%)
- TP1: 159.75 · TP2: 159.52 · R:R ~1:2

### Риски
- Отскок после крупного long liq (mean reversion 5–15m)
- Mid-cap низкая ликвидность — проскальзывание
- Без HTF — возможен внезапный пробой вверх на новостях

## Пример 2: Post long-liq cluster · типичный SHORT continuation

**Паттерн:** цена↓, long liq spike 100K+, через 10m цена +0.5% отскок, CVD всё ещё↓, OI↑ при слабом отскоке.

### Вердикт
**SHORT на ретесте** · **7/10**

- Liq = fuel spent, не разворот
- OI↑ + цена не может выше pre-liq high = `aligned_short` / `shorts_building`
- Вход: 50–61.8% retracement отскока от liq low
- Стоп: над high отскока
- `continuation_risk=True` в терминах бота

## Пример 3: Squeeze setup

**Паттерн:** цена↑, OI↓, CVD↑, short liq spikes на хаях.

### Вердикт
**NO NEW SHORT** · рассмотреть **LONG pullback** · **5/10**

- `squeeze_risk` — шорты закрываются
- Short только после structure break (LL на 5m)
- Long: откат к broken resistance + CVD holds

## Пример 4: WAIT — смешанные панели

**Паттерн:** цена в mid-range, CVD flat, OI flat, liq нет.

### Вердикт
**WAIT** · **3/10**

- Нет edge; ждать пробой range + OI expansion или новый liq cluster
- Не торговать «на угадайку» в центре боковика
