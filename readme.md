# Bybit_bot — Crypto Futures Scanner

Telegram-бот для сканирования pump/dump сигналов на фьючерсах **Binance** и **Bybit** (USDT perpetual). Анализирует Open Interest, цену и объём в реальном времени через официальные API v5.

## Возможности

- 🟢 **LONG** / 🔴 **SHORT** сигналы с уведомлениями в стиле торгового скринера
- 📈 **Open Interest screener** — приток/отток капитала за 1–30 минут
- 💹 **Pump / Dump** — резкие движения цены + OI
- 🔗 **CoinGlass** — ссылка на анализ каждой монеты
- 🔊 **Сила сигнала 1–10** — ранний (1) / поздний (10) вход
- 🔥 **Приоритет** — звук и закреп для score ≤ 2
- ⏸ **Пауза** — кнопка «Стоп» в конце дня, «Старт» на следующий
- ⚙ **Настройки в Telegram** — кнопки + команда `/set`
- 🏆 **Топ-N** монет по объёму
- 📊 **Разные пороги** для Binance и Bybit

## Быстрый старт

```bash
cp .env.example .env
# Заполните TELEGRAM_TOKEN и TELEGRAM_ADMIN_ID

docker compose build
docker compose up -d
```

В Telegram: `/start` → `/test` → настройте пороги через **🔧 Настройки**.

## Команды (кратко)

| Команда | Описание |
|---------|----------|
| `/test` | Тестовые LONG + SHORT |
| `/pause` / `/resume` | Стоп / старт уведомлений |
| `/status` | Текущие настройки |
| `/set help` | Точная настройка порогов |
| `/history 20` | История сигналов |

## Кнопки в чате

- **⏸ Стоп** / **▶️ Старт** — пауза сигналов
- **📊 Биржи** — вкл/выкл Binance, Bybit
- **🔧 Настройки** — пороги OI, цены, период

## Стек

- Python 3.11, asyncio
- python-telegram-bot
- Bybit / Binance WebSocket + REST API v5
- Redis (опционально)
- Docker Compose

## Важно

Бот **не гарантирует прибыль** — он быстро находит аномалии на рынке. Решение об открытии сделки принимаете вы.

API-ключи бирж для сканирования не требуются.
