Техническое задание
Crypto Futures Scanner
Общая информация
Необходимо разработать полноценную профессиональную систему анализа криптовалютного фьючерсного рынка с возможностью обнаружения ранних сигналов для открытия Long и Short позиций.
Важно понимать, что система не должна пытаться предсказывать рынок или гарантировать прибыльность. Ее задача — максимально быстро анализировать поступающие данные с бирж, обнаруживать аномальную активность и уведомлять пользователя. Окончательное решение об открытии сделки всегда принимает человек.
Проект разрабатывается исключительно для личного использования.
Основная задача
Создать высокопроизводительную систему, которая в режиме реального времени анализирует абсолютно все USDT Futures на биржах Binance Futures и Bybit Futures.
Количество инструментов постоянно изменяется (примерно 300+ торговых пар на каждой бирже), поэтому список монет должен обновляться автоматически.
Все монеты анализируются одновременно.
Все данные должны обновляться максимально быстро (целевое время обновления — около 1 секунды, насколько это позволяют WebSocket API бирж).

Scanner №3
Open Interest Scanner
Это главный модуль проекта.
Он должен анализировать изменение Open Interest по каждой монете.
Важно.
Система должна анализировать именно изменение глобального Open Interest по инструменту, отражающее приток или отток капитала, а не отдельные позиции конкретных трейдеров, поскольку такие данные биржи не предоставляют.
